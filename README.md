# 个人博客 & 技术简历网站

欢迎来到我的个人博客和技术简历展示网站！这里是我分享技术心得、记录学习历程的地方。

## 🌟 网站特色

### 📋 简历展示
- 完整的个人简历信息
- 项目经历展示
- 技术栈可视化
- 工作经验介绍

### 📚 知识分享
- 技术博客文章
- 学习笔记整理
- 开发经验分享
- 最佳实践总结

### 🎨 设计特点
- 现代简洁的设计风格
- 温和的色彩搭配（避免鲜艳颜色）
- 响应式布局设计
- 优秀的用户体验

## 🚀 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **样式**: CSS自定义属性, Flexbox, Grid
- **字体**: Inter字体家族
- **图标**: Font Awesome 6
- **部署**: GitHub Pages

## 📁 项目结构

```
简历/
├── index.html              # 主页
├── blog.html              # 博客列表页
├── styles/
│   ├── main.css           # 主要样式文件
│   └── blog.css           # 博客页面样式
├── scripts/
│   ├── main.js            # 主要JavaScript
│   └── blog.js            # 博客页面功能
├── README.md              # 项目说明
└── 简历_优化版.md          # 原始简历内容
```

## 🌈 主要功能

### 主页功能
- **响应式导航栏**: 固定顶部导航，带有平滑滚动效果
- **Hero区域**: 个人介绍和核心信息展示
- **关于我**: 详细的个人背景和技术栈
- **精选项目**: 重点项目的卡片式展示
- **技术博客**: 最新文章预览
- **联系方式**: 多种联系方式展示

### 博客页面功能
- **文章过滤**: 按技术分类筛选文章
- **智能搜索**: 支持标题、内容、标签搜索
- **特色文章**: 重点文章的大卡片展示
- **文章统计**: 实时显示文章数量和阅读量
- **订阅功能**: 邮箱订阅技术动态
- **键盘导航**: 支持快捷键操作

### 用户体验优化
- **平滑动画**: Intersection Observer实现的滚动动画
- **键盘支持**: 全面的键盘导航支持
- **加载优化**: 字体和资源的预加载
- **SEO友好**: 完整的meta标签和语义化HTML

## 🎯 设计理念

参考了创意开发者作品集的现代设计风格，但采用了更加温和的色彩方案：

### 色彩搭配
- **主色调**: `#2c3e50` (深蓝灰)
- **次要色**: `#3498db` (温和蓝)
- **强调色**: `#7f8c8d` (中性灰)
- **背景色**: `#ffffff` (纯白) / `#f8f9fa` (浅灰)

### 设计原则
- **简洁明了**: 避免过度设计，突出内容本身
- **层次分明**: 合理的信息架构和视觉层次
- **一致性**: 统一的设计语言和交互模式
- **可访问性**: 良好的对比度和键盘导航支持

## 📱 响应式设计

网站采用移动优先的响应式设计：

- **桌面端**: 完整的多列布局和丰富交互
- **平板端**: 适配中等屏幕的网格布局
- **移动端**: 单列布局和触控优化

## 🛠️ 本地开发

1. 克隆仓库到本地
2. 使用Live Server或其他静态服务器打开`index.html`
3. 开始开发和修改

## 📝 内容更新

### 添加新文章
1. 在`blog.html`中添加新的文章卡片
2. 设置正确的`data-category`属性
3. 更新文章统计信息

### 修改项目信息
1. 编辑`index.html`中的项目卡片
2. 更新项目链接和描述
3. 同步更新技术标签
